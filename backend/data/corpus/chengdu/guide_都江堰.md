# 都江堰
来源: https://zh.wikivoyage.org/wiki/都江堰
类型: 维基导游（交通/住宿/景点实用指南）

都江堰市 ，旧称 灌县 ，是 成都市 代管之县级市，位于成都平原西北部，以都江堰水利工程而得名，全市幅员面积1208平方公里，辖13个镇、5个街道、1个乡，1个省级经济技术开发区，户籍人口约61.88万人。
都江堰是国际旅游名城，拥有举世瞩目的世界文化遗产—都江堰·青城山，是四川大熊猫栖息地世界自然遗产的重要组成。
