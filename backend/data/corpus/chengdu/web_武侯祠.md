# 武侯祠
来源: https://zh.wikipedia.org/wiki/武侯祠

武侯祠 ，是为了纪念 三国 蜀汉 丞相 诸葛亮 而兴建的祭祀建筑。其中 勉县武侯祠 为 刘禅 所立的官祠，始建年代最早； 南阳武侯祠 由诸葛亮故将 黄权 建立，始于魏晋年间 ； 成都武侯祠 ，始建于 成汉 年间，君臣合祭，最为知名。
成都武侯祠 ，位于 四川省 成都市 武侯区
清溪武侯祠 ，位于 四川省 雅安市 汉源县
凉山武侯祠 ，位于 四川省 凉山彝族自治州 越西县
祁山武侯祠 ，位于 甘肃省 陇南市 礼县
木门道武侯祠 ，位于 甘肃省 天水市 秦州区
五泉山武侯祠 ，位于 甘肃省 兰州市 城关区
勉县武侯祠 ，位于 陕西省 汉中市 勉县
城固武侯祠 ，位于 陕西省 汉中市 城固县
五丈原诸葛亮庙 ，位于 陕西省 宝鸡市 岐山县 蔡家坡镇
南阳武侯祠 ，位于 河南省 南阳市 卧龙区
黄陵庙武侯祠 ，位于 湖北省 宜昌市 夷陵区
石鼓书院武侯祠 ，位于 湖南省 衡阳市 石鼓区
新阳武侯祠 ，位于 福建省 漳州市 云霄县
翠屏武侯祠 ，位于 广西壮族自治区 桂林市 阳朔县
保山武侯祠 ，位于 云南省 保山市 隆阳区
嵩明武侯祠 ，位于 云南省 昆明市 嵩明县
通海武侯祠 ，位于 云南省 玉溪市 通海县
^ 据《明嘉靖南阳府志校注》和清康熙《龙岗志》载，诸葛亮殒没五丈原后，其故将 黄权 曾率族人在南阳卧龙岗建庵祭祀，时称“诸葛庵”。 晋 永兴 年间，镇南将军刘弘“镇宛”讨伐张昌时“观亮故宅”并“立碣表闾”。
